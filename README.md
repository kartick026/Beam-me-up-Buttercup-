# React Shooter Game

A polished 2D Top-Down Shooting Game built with React and HTML5 Canvas.

## Features
- **Player Movement**: WASD for desktop, Virtual Joystick for mobile.
- **Shooting**: Mouse aiming/clicking, Touch shooting.
- **Enemies**: Normal (Red) and Fast (Pink) enemies with homing logic.
- **Boss Fights**: Boss appears every 12 score points with radial attacks.
- **Powerups**: Rapid Fire, Double Shot, Shield.
- **Audio**: Synthesized sound effects using Web Audio API (no assets needed).
- **Responsive**: Fullscreen canvas adapting to screen size.

## Installation

1.  Navigate to the project directory:
    ```bash
    cd react-shooter-game
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open the link provided in the terminal (usually `http://localhost:5173`).

## Controls
- **Move**: WASD / Arrow Keys / Left Virtual Joystick
- **Aim**: Mouse Cursor
- **Shoot**: Left Click / Spacebar / Right Touch Zone
- **Dash**: Shift / Dash Button (Mobile)
- **Pause**: P Key

## Technologies
- React
- HTML5 Canvas
- Web Audio API
- Vite
